你之所以进步慢，是因为你不敢扔掉你的救生圈

发表于 2018-11-21 | 更新于 2018-11-22 | 分类于 说说 | 阅读次数：

胡立阳是全球最受欢迎的华人投资理财专家，他有个称号——「亚洲股市教父」。

他小时候很喜欢刺激、冒险，五六岁时就学会了游泳，自认为泳技高超。

十三岁那年决定到大海里去大显身手，他抱着一个很大的救生圈下了海，由于那时是退潮，他没料到海浪的力量有那么大，很快就把他带出去了，不一会儿，就已经离警戒线很远。

他使劲地想游回岸边，但由于是退潮，加上套在身上的救生圈实在太重，根本就游不动。自己被海水越带越远，岸上的人越来越小，天色已晚，没有人来救自己，心想：难道自己要葬身大海喂鱼了吗？

他现在有两个选择：要么抱着救生圈，等着奇迹出现，要么丢掉救生圈，靠自己的力量拼命游回去。可是丢掉救生圈，如果游不回岸上，又没人来救自己，怎么办？但不丢掉救生圈，动不了，就只能坐以待毙！

在刹那间，他做出了一个生死攸关的决定——将救生圈丢掉。

救生圈刚被扔掉之后，立刻就被浪带走了。他现在没有别的选择，只能靠自己的力量，拼命的往岸边游。当他感到无力时，他不断鼓舞自己来驱除内心的恐惧。虽然仍游得辛苦，但陆地渐渐触目可及，就这样拼命游了两个小时，终于游到了岸边，当手碰到沙滩的那一刻，他瘫倒了，同时喜极而泣，劫后重生的感觉令他永生难忘。

他后来给朋友谈起这个事情，大家都说他疯了，没有了救生圈，就没了依赖，没有了希望。但胡立阳清楚，自己是对的。

这次生死攸关的经历，对胡立阳日后的性格产生了巨大的影响。他摆脱了依赖心理，做事更加决断。

后来，他进了华尔街，很快，他就成为了华尔街股市神童、华尔街顶级大师、全球最受欢迎的华人投资理财专家。

故事比较极端，在这样的情况下，大部分的人应该是淹死了……不过这个故事对我的触动很大。

我们回头想一下，是不是也遇到过类似的问题呢，很多事情，我们惧怕扔掉所谓的「救生圈」，导致自己进步很慢。

我们一生中有太多太多的救生圈，只是我们不曾发现，而我，也是前段时间突然醒悟，从小到大，我一直在给自己套救生圈，乐此不疲，为了让自己轻松点、安全点。这种观念在我心里，一直没有去除。

很多人学不会游泳，是因为总套着一个救生圈，小时候去游泳，我喜欢拿着个汽车车胎，我最喜欢趴在这个「救生圈」上，同龄人都学会了游泳，我还不会。那时候，汽车车胎我的救生圈，把生的希望寄托给它，这个寄托是让我们不能很快学会游泳的罪魁祸首。

上大学的时候，我从来不喜欢做笔记，总想着可以借别人的，我室友一直是我们班第一名，还是我老乡，我一直都是抄他的笔记和作业，考试也不怎么认真复习，想着能不能看看隔壁的，但是因为抄袭水平不高，或者老师监考太严，导致没有得逞，也因此考过同一学期 3 门必修课 181 分都没挂的好成绩，当时发了朋友圈说这事儿，一个同班妹纸看到了，赶紧给我发消息，说「哇，你考 181 啊，我刚好考 180，都没挂，哈哈哈哈哈哈」。

不得不说，把责任放在别人身上，让我自己行走起来确实轻松了不少，但是步伐却如此缓慢。

工作之后好了吗？其实并没有，步入社会之后的很长一段时间，我其实还在打着救生圈的主意。

我一直很感激工作中带我的领导和同事，不知道是不是我运气好，碰到的人都是对我很好的。

刚毕业实习的时候，我只知道 Java 的名字而已，第一个导师就手把手教我写了很多代码，后来有了经验，仍然不去独自思考问题，有时候自己离答案只有一个纸墙之隔，就去问，而不是先思考出来方案，说出方案的优缺点，让领导去做选择；不敢去承担，总想把责任放在别人身上，觉得自己搞不定还有领导呢，总想让自己轻松一点。

后来我明白了，时刻打着救生圈的主意，其实害的是自己，长进很慢，刚毕业的时候还会有人带带你，后面搞不定很可能就会直接抛弃你，社会不会让你一直呆在救生圈里，如果你抛弃不了救生圈，社会就会抛弃你。

丢掉吧，丢掉一切阻止你前行的救生圈。有了它，你就总会在心里留一个后退的空间。破釜沉舟，勇往直前才是王者风范。

现在我这种思维就没有了，如果同事和领导帮助，我会一边感恩，一边警醒自己要做的更好。
