如何让豆瓣和 IMDb 互相显示评分

发表于 2018-06-27 | 分类于 工具 | 阅读次数：

不少人看电影时喜欢参考豆瓣电影和 IMDb 的评分，每次都会先打开一个，然后再打开另一个，今天就来说说怎么在豆瓣电影和 IMDb 互相显示评分，电影党必备。

没错，我说的就是 Tampermonkey（油猴插件）

Tampermonkey 是一款免费的浏览器扩展和最流行的用户脚本管理器（插件），支持Chrome、Microsoft Edge、Safari、Opera Next、Firefox、Dolphin Browser、UC Browser 等浏览器平台。

首先去 Tampermonkey 官网(https://tampermonkey.net/ )下载这个插件，不过需要科学上网，你懂得。如果能正常下载，就在应用商店内下载就好。

不能的，这里推荐一个网站， https://www.crx4chrome.com/ ，可以下载几乎所有的 Chrome Store 的插件，在 Crx4Chrome 搜索 Tampermonkey 或者 访问这个 https://www.crx4chrome.com/extensions/dhdgffkkebhmkfjojejmpbldmpobfkfo/ 下载

打开 Chrome 扩展程序页面，将下载好的 crx 文件拖拽到页面即可完成安装。右上角会有望远镜小图标。

左键点击望远镜图标，进入管理面板 -> 已安装脚本，默认是没有任何脚本的。

点获取一些脚本，由于需要科学上网，这个也是打不开的，这个页面上展示的是几个下载源，OpenUserJS，GreasyFork，Userscripts.org，GitHub/Gist

这里面最好用的就是 Greasy Fork https://greasyfork.org/zh-CN

进入 Greasy Fork 搜索 MoreMovieRatings 或者直接进入 https://greasyfork.org/zh-CN/scripts/7687-moremovieratings

点击安装这个脚本，如果弹出安装页面，在点一下安装，有的脚本不弹就安装了，自己看一下。

再进入你的管理面板，会发现已经安装好了

这时候打开豆瓣电影，随便选一部，比如 「肖申克的救赎 The Shawshank Redemption」

你会发现，两个评分都会显示出来了。

同样的，你打开 IMDb，也会发现两个评分也都会显示出来。

其实这只是一个入门，告诉你油猴该怎么用，里面有更广阔的天地，就由你自己来发现吧，你会发现很多有趣的东西。后面我发现好玩的也会分享出来，期待后续的话，可以关注一下我。
