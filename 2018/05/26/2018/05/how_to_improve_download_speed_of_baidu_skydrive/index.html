如何把百度网盘下载速度提高 100 倍，我推荐这个下载工具

发表于 2018-05-26 | 更新于 2018-06-09 | 分类于 工具 | 阅读次数：

应该有很多小伙伴平时的文件、资源都会放到百度网盘上面，或者从别人网盘上找到各种资源，但是当你下载的时候你就发现百度云是以这种速度龟速下载。对，每秒几十 KB，小的文件还好，如果稍微大点的软件，就得先去玩一天，慢慢等着了。

反正我已经懒得喷百度了，封了很多第三方下载工具，意思就是让你开 VIP，而我又不想开，因为并不常用百度云下载，索性就去网上搜罗一些工具，刚好发现了这样一款优秀的全速下载工具。多亏有我们的开源社区以及广大网友啊，又想到了新方式。这里我就分享这款优秀的下载工具，让你告别下载缓慢的烦恼。

废话不多说，先上图，看效果。可以看到下载速度接近 3MB 每秒，这个速度是依赖你现有网络的，我的网络不太快，所以也是接近全速下载了。

这个工具的名字是 proxyee-down，是在 GitHub 上发现的，工具的地址是

https://github.com/proxyee-down-org/proxyee-down

上面有安装教程，以及问题解答，一般看这个教程就能搞定了，如果还有问题，可以加他们的 qq 群，以上地址里面都有。

考虑到这个步骤有一些坑，尤其是没有技术背景的看文档可能还是有点困难，我这里把我的步骤讲一下，希望能够尽可能帮助到大家少走弯路。PS：我的系统是 Mac OS，浏览器是 Chrome，不是这个系统和浏览器的仅做参考，可以去上面安装教程里面找到对应的其他系统安装方式。

步骤：

1、下载这个工具

地址：

https://pan.baidu.com/s/1fgBnWJ0gl6ZkneGkVDIEfQ#list/path=%2F

提取码: d92x

里面有对应的版本，不同的操作系统下载对应的就好了，proxyee-down-x.xx-mac.zip, 现在最新的是 2.54，比如我下载的是 proxyee-down-2.54-mac.zip

2、解压并运行

讲刚才下载好的 proxyee-down-2.54-mac.zip 解压到任意目录，双击 start 运行软件，由于安装证书和切换代理需要 root 权限，所以在运行软件会提示输入用户密码，输入完按回车确认即可。

启动成功之后会弹出成功提示，同时启动了这个工具。

同时电脑上会有如下的这个工具图标，点开证书目录，其实就是刚才目录里面的 proxyee-down-2.54/main/ssl/ca.crt

双击安装证书，添加可信任。

3、打开你想下载的百度网盘链接

你会发现多出来一个 proxyee-down，如果没有出现，检查以上步骤是不是有问题，最后还是不出现，去该工具地址看常见问题列表 https://github.com/proxyee-down-org/proxyee-down/blob/master/.guide/FAQ.md，看是否你那边需要装浏览器插件，如果装了还是有问题，可以去 qq 群问问题，或者留言给我。

4、选中你要下载的文件，点下载按钮

点了下载按钮之后就会跳转到这个工具，会有个弹出框，如下图，选择你的下载路径，点击创建，就开始下载啦。

这样你就告别百度云的龟速下载了。
